# Kaenx.Konnect
=================

[Nuget Package](https://www.nuget.org/packages/Kaenx.Konnect/) is availible.

Kaenx.Konnect is a library to connect to a KNX Interface.  
Its supports IP Tunneling, USB* and IP Routing*.


*Not implemented yet
